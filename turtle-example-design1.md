## python design1

Python turtle example: design1

```python
from turtle import 
speed(0) # sets the speed of drawing to 0, which is the fastest
pencolor('white') 
bgcolor('black')

x = 0 
up()
rt(45) 
fd(90) 
rt(135) 

down()
while x < 120: 
    fd(200)     
    rt(61)
    fd(200)
    rt(61)
    fd(200)
    rt(61)
    fd(200)
    rt(61)
    fd(200)
    rt(61)
    fd(200)
    rt(61)

    rt(11.1111) 
    x = x+1 

exitonclick() 

```
