## python my turtle

Python turtle example: my turtle

```python
from turtle import *
import turtlehelp

window = Screen()

amy = Turtle()
amy.speed(100)
amy.espiral()

amy.comienzo()

amy.escalera()

amy.comienzo()

amy.fillcolor('pink')
amy.estrella(7,3)

amy.comienzo()

for j in range(3,8):
    amy.poligono(j)

amy.comienzo()

window.exitonclick()



```
