## python turtle simple

Python turtle example: turtle simple

```python
from turtle import *

scale = 0.05 # choose 1, 0.05 
power = 1 # nice one: 1, 0.9, 2, 0.8

speed(0) # 0 = fastest 

for i in range(10000000):
    forward(scale * i)
    right(i**power)

```
