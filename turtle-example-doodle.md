## python doodle

Python turtle example: doodle

```python
from turtle import *                    
pencolor('green') 
bgcolor('black') 
#speed(0) #Uncomment this to draw super fast
c = 0 
penup() 
rt(45) 
fd(90) 
rt(135) 
pendown()
while c < 100:
               
    fd(200)     
    rt(80)
    fd(200)
    rt(80)
    fd(200)
    rt(80)
    fd(200)
    rt(80)
    fd(200)
    rt(80)
    rt(11) 
    c = c+1 
            

exitonclick() 


```
