## python turtle2

Python turtle example: turtle2

```python
from turtle import *
shape("turtle")
色 = ["orange", "red", "black", "purple", "blue"]
for 数だけ in range(5):
    color(色[数だけ])
    width(40)
    forward(300)
    left(144)
forward(-200)
for 数だけ in range(1,6):
    circle(100)
    left(72)

forward(200)
for 数だけ in range(1,5):
    forward(200)
    right(90)


done()

```
