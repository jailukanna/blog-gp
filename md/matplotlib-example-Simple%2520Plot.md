## python Simple%2520Plot

Python matplotlib example: Simple%2520Plot

```python
# coding: utf-8
from pylab import *

t = arange(0.0, 2.0, 0.01)
s = sin(2*pi*t)
plot(t, s)

xlabel('time (s)')
ylabel('voltage (mV)')
title('About as simple as it gets, folks')
grid(True)
show()


```
