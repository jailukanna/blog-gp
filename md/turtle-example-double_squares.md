## python double squares

Python turtle example: double squares

```python
from turtle import *
#Line 3-7: draw first square
forward(150)

for i in range(3):
    right (90)
    forward(150)

#Line 10-14: draw second square 
forward(150)

for i in range(3):
    right (90)
    forward(150)

exitonclick()

```
