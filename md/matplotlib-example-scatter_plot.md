## python scatter plot

Python matplotlib example: scatter plot

```python
# a scatter plot just shows one point for each dataset entry

x = np.random.rand(100)
y = np.random.rand(100)
plt.title("Simple scatter plot")
plt.scatter(x,y)
plt.show()
```
