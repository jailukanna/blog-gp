## python plot

Python matplotlib example: plot

```python
import matplotlib.pyplot as plt

fig = plt.figure(figsize=(10,10))
ax = fig.add_axes([0.2,0.2,0.7,0.7])
ax.plot()
# ax.set_ylim()
# ax.set_xlim()
# ax.legend()
ax.set_xlabel('',fontsize=15)
ax.set_ylabel('',fontsize=15)

```
