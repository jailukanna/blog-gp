## python creneau

Python turtle example: creneau

```python
from turtle import *
for a in range(1,5):
    for b in range(1,6):
        forward(10)
        left(90)
        forward(10)
        right(90)
        forward(10)
        right(90)
        forward(10)
        left(90)
    forward(10)
    right(90)

done()


```
