## python Mesmerizing

Python turtle example: Mesmerizing

```python
from turtle import *
colors = ['red', 'purple', 'blue', 'green', 'yellow', 'orange']
speed(10)
for x in range(360):
    pencolor(colors[x % 6])
    width(x / 1 + 1)
    forward(x)
    left(10)
    right(-5)
    
    
    

```
