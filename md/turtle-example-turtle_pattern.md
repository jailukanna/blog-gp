## python turtle pattern

Python turtle example: turtle pattern

```python
from turtle import *
colors=["red","blue","green","yellow","purple"]
for i in range(100):
    pencolor(colors[i%5])
    width(i/100+1)
    forward(i)
    left(59)
exitonclick()


```
